API Testing Framework in Java with RestAssured, TestNG, and Allure Reporting

This project represents a framework for API testing, built in Java using RestAssured, TestNG, and Allure Reporting. The framework is designed to be easy to extend and use in real projects, and it includes full Allure integration, POJO models, logging, and a CI/CD pipeline configured with GitHub Actions.

Technologies Used
Java 17 RestAssured 5 TestNG 7 Allure TestNG and Allure RestAssured Maven Jackson for serialization and deserialization SLF4J for logging.

How to Run the Tests
Run the tests: mvn clean test
Generate the Allure report locally: allure generate target/allure-results --clean allure open allure-report

Allure Report Online
The Allure report is automatically generated by GitHub Actions and published on GitHub Pages. You can access it here: https://victoria198611.github.io/api-testing-framework/

CI/CD Integration
The project is integrated with GitHub Actions. On every push to the main branch:
– the tests are executed – the Allure report is generated – the report is published online
The workflow runs the mvn clean test command and automatically processes the Allure files.

environment.properties File
This file is automatically included in the Allure report.
Example:
base.url=https://reqres.in environment=local api.version=v1 tester=Victoria

Useful Commands
List Allure results: ls target/allure-results
Delete Allure results: rm -r target/allure-results
